## 如何在本地编辑并运行

1. 安装 `hugo-extended`
2. `git clone https://github.com/fvtt-cn/docs` 克隆仓库到本地 
3. 在本地的仓库目录下执行 `hugo server -D`
4. 打开输入的地址，即可本地实时预览站点效果
